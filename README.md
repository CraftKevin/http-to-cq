# 简介
将酷Q与PhP对接起来，并通过Http和WebSocket交互调用酷Q API

文件说明
--------
`cc.1sls.CtPe.cpk` - 运行所需插件

`sdk.php` - 酷Q函数文件，如果使用WebSocket交互(动态API)则不可缺少
使用前需将ip和port与插件设定一致

`demo.php` - 模板文件，可以在此基础上继续编写代码或整合至原有代码
